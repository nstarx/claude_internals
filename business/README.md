## Analysis Modes

**Three adaptive interaction strategies:**

1. **Discussion Mode** - Collaborative multi-perspective analysis
2. **Debate Mode** - Adversarial stress-testing through structured disagreement
3. **Socratic Mode** - Question-driven strategic thinking development
