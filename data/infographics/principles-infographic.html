Principle Hierarchy

1. Minimize Context Pollution

  • Keep only task-relevant information
  • Proactively prune outdated context
  • Quality over quantity always

2. Front-Load Critical Info

  • Most important info first
  • Structure for immediate clarity
  • Reduce cognitive scanning

3. Use Tools for Retrieval

  • Don't pre-load everything
  • Retrieve on demand
  • Just-in-time information

4. Persist Patterns

  • Document recurring patterns
  • Build knowledge over time
  • Reduce rediscovery cycles

Apply principles in sequence for optimal context management

Key Takeaways

  • Each principle builds on the previous ones
  • Tools exist to implement each principle
