Tool Workflow
Tools work together to manage context throughout development

Discovery Tools

  • Glob: Find files by pattern
  • Grep: Search content
  • Read: Load file content

Modification Tools

  • Edit: Precise changes
  • Write: Create files
  • Bash: System commands

Analysis Tools

  • LSP: Code intelligence
  • Task: Subagent delegation

External Tools

  • WebFetch: Fetch URLs

Tool Best Practices

  • Read before Edit: Always understand context first
  • Use Glob over Bash find: Optimized file search
  • Parallel calls: Run independent operations together
  • Prefer Edit over Write: Preserve existing content
