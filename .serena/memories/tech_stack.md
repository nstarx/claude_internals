# Tech Stack

## Frontend
- **HTML5**: Semantic markup for documentation viewer
- **CSS3**: Modular stylesheets (main, components, navigation, artwork, responsive, etc.)
- **JavaScript (ES6+)**: Dynamic content loading and UI interactions

## Key Libraries
- **PrimeIcons**: Icon library (https://unpkg.com/primeicons/primeicons.css)
- **Marked.js** (likely): For markdown rendering in browser

## Content
- **Markdown**: All documentation written in markdown format
- **JSON**: Data files for formulas and structured content

## Deployment
- **GitHub Pages**: Static site hosting
- **No build step**: Direct deployment of HTML/CSS/JS files
