# Project Overview

## Purpose
**SuperClaude Framework** - A comprehensive framework for enhancing Claude Code with advanced behavioral modes, MCP server integrations, and business analysis capabilities.

## Key Features
- **Documentation Framework**: Extensive documentation system covering behavioral modes, MCP integrations, and business analysis
- **Web Viewer**: Interactive HTML-based documentation viewer deployed to GitHub Pages
- **Token Optimized**: All documentation optimized for efficient context loading (max 600 lines per file)
- **Modular Architecture**: Clean separation between content (markdown), presentation (HTML), styles (CSS), and behavior (JavaScript)

## Live Site
https://nstarx.github.io/claude_internals/

## Project Type
- Static documentation website
- GitHub Pages deployment
- No build process or package manager
- Pure HTML/CSS/JavaScript with markdown rendering
